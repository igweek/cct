## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
在 Settings (设置) -> Environment Variables (环境变量) 中添加：
NOTION_API_KEY: 您的 Notion Secret
NOTION_DATABASE_ID: 您的 Database ID
